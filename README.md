# Tarea: Input, Output y Data Types en C++

Nombre: Viviana Sofia Feijoo Ochoa  
Curso: 2684

## Descripción

Este programa solicita al usuario ingresar su nombre, edad, estatura, peso e inicial del apellido. 
Luego muestra en pantalla todos los datos ingresados utilizando entrada (cin) y salida (cout).

Se utilizan diferentes tipos de datos:
- string
- int
- float
- double
- char

## Compilación y ejecución

Para compilar:
g++ deber1.cpp -o deber1

Para ejecutar:
./deber1
